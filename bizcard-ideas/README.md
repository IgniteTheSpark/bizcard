# BizCard Ideas 模块

本目录存放 **Ideas 模块** 相关的产品说明、PRD 与技术方案。

## 模块概要

- 用户可**选择一条或若干条 Notes**（会议或闪念），由 **AI 生成一份 Idea**（如方案、报告、调研）。
- 生成的 Idea 在 **Ideas 模块** 中查看与管理，并**关联回来源 Notes**；在 Note 详情中可展示「关联的 Ideas」。

## 文档

- **[IDEAS_MODULE_SPEC.md](./IDEAS_MODULE_SPEC.md)**：概念、流程、内容模型、与 Notes/Calendar/Reminder 的关系及后续可展开点。
