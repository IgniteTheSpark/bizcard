## 🎯 轻量版本建议

基于复杂度分析，可以考虑以下轻量化方向：

### 方案 A：简化 Calendar 页面
- **保留**：日视图（周日期条 + Reminders 列表）
- **移除**：月视图的展开交互（Lark 风格）
- **简化**：月视图改为简单的月历网格，点击直接跳转到日视图
- **节省工作量**：约 30-40%

### 方案 B：简化左滑操作
- **保留**：点击卡片显示操作菜单（Popover）
- **移除**：左滑手势
- **节省工作量**：约 20%

### 方案 C：简化 Date Picker
- **保留**：基础日期选择器
- **简化**：统一使用一种模式，不区分日视图/月视图
- **节省工作量**：约 15%

### 方案 D：组合方案（推荐）
- **简化 Calendar**：只保留日视图，月视图改为简单网格
- **简化操作**：用 Popover 替代左滑
- **保留核心**：Reminders Hub、Meeting Reminders、Contact Activity
- **预计节省**：约 40-50% 开发工作量
