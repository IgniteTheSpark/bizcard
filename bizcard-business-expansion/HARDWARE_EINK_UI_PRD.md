# MeCard 硬件 E-ink 墨水屏交互与呈现契约设计

## 一、 定位与核心设计哲学

在 MeCard 战略中，硬件设备从一个单一的“后置录音采集器”，蜕变为用户的**“前置商务伴随者 (Copilot)”**。

### 1.1 核心设计哲学：手机大脑，端侧呈现
为了解决端侧算力不足、发热、以及对极致长续航的要求，我们采用以下架构：
- **大脑在手机/云端**：所有的复杂计算（ASR、NLP 理解、大模型推理、日历查询）均在手机 App 侧或云端完成。
- **排版在手机**：手机 App 基于大模型的输出，将内容排版渲染成一张完美的 E-ink 兼容图片（或结构化局部刷新指令）。
- **显示在端侧**：硬件设备通过低功耗蓝牙 (BLE) 仅接收最终的渲染结果并控制 E-ink 屏幕刷新。

### 1.2 E-ink 交互原则
墨水屏的物理特性决定了其交互不能像手机一样：
1. **摒弃高频滚动**：避免瀑布流或频繁的滑动操作。
2. **块状刷新 (Block Refresh)**：采用翻页、卡片替换、或局部区域（如气泡）更新。
3. **极简排版**：高对比度、大字体、突出关键数据（时间、人名、Action）。

## 二、 核心场景交互流 (Interaction Flow)

### 场景一：Morning Briefing (每日晨报)
这是占用用户桌面或随身携带的常态化待机界面。

- **触发机制**：每日早晨（如 7:00 AM）手机端定时任务触发，或用户拿起设备时。
- **数据来源**：调用 MCP 聚合多方 Calendar，提取今日高优 Contacts 动态。
- **手机端处理**：渲染一张 `Dashboard` 图片。
- **E-ink 呈现排版**：
  - [顶部]：当前日期与时间（大字号）。
  - [中部左侧]：今日核心日程 Timeline（如 `10:00 飞书会-产品过会`, `14:00 面见 Kevin`）。
  - [中部右侧]：Ask Agent 提取的今日关键提醒（如 `记得带上昨晚下载的 RWA 研报`）。
  - [底部]：一句话激励或天气。
- **刷新频率**：极低（仅在有新日程变更或手动翻页时全刷）。

### 场景二：销售/商务 Copilot (提词器)
这是在面对面或电话会议中，硬件作为“最强辅助”的场景。

- **触发机制**：会议录音进行中，手机端 ASR 持续流式识别。
- **数据来源**：手机端 Agent 实时监听对话流，匹配客户异议或捕捉到关键痛点。
- **手机端处理**：一旦命中策略（如客户提到竞品 A），大模型迅速生成三条“应对策略”并渲染为卡片图。
- **E-ink 呈现排版**：
  - 全屏弹出 `🚨 应对建议` 卡片。
  - 使用极其醒目的加粗字体展示关键话术（如 `强调我们系统的本地化沙箱隔离能力，这是竞品A没有的`）。
- **刷新频率**：中等（事件触发式局部刷新）。

### 场景三：同声传译 (气泡式刷新)
跨国会议或外语交流时的便携翻译机。

- **触发机制**：开启翻译模式。
- **手机端处理**：流式 ASR + 机器翻译，按“句”或“完整意群”切分。
- **E-ink 呈现排版**：
  - 上下分屏布局：上方为识别到的原文，下方为翻译后的译文。
  - **交互创新**：绝对不使用跑马灯滚动。而是采用“段落气泡替换”的方式。新的一句翻译出来后，直接覆盖掉旧的一句（局部快刷），保证视觉不眩晕。

### 场景四：闪念备忘 (Voice Memo -> Screen)
随时随地的灵感捕捉面板。

- **触发机制**：用户按住硬件上的物理快捷键，说出一句话（如“下周一早上提醒我给行政交报销单”）。
- **手机端处理**：音频传给手机，手机 ASR 识别后，不仅存入 App 的 Reminders，同时将这句话转为大字报图片。
- **E-ink 呈现排版**：
  - 屏幕中央超大字号展示刚才说的话，类似于一张贴在电子屏幕上的“便利贴”。
  - 存在 15 分钟后自动恢复为默认的 Dashboard。

## 三、 蓝牙通讯契约 (App <-> Hardware)

为了支持上述功能，App 与硬件之间的蓝牙通讯协议主要包含以下指令：

1. `CMD_PUSH_FULL_FRAME`: 推送一张全屏 E-ink 格式图片并要求全局慢刷（用于 Morning Briefing 切换）。
2. `CMD_PUSH_PARTIAL_BLOCK`: 推送一个局部坐标块的图像数据并要求快速局刷（用于 Copilot 提词卡片弹出）。
3. `CMD_CLEAR_SCREEN`: 清屏指令。
4. `CMD_WAKE_UP`: 唤醒指令（手机主动唤醒低功耗休眠的硬件以推送重要消息）。
