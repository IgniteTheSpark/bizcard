## 关键口径速查

- **断连策略**：耳机录音模式下，任一经典蓝牙链路断开 → 录音终止（不做无缝降级）
- **命名**：统一使用「设备控制台」，不混用「设备详情」/「硬件管理页」
- **Phase 1 命名**：「连接手机音频链路」，不使用「系统授权」
- **强接管**：耳机已连接时，所有录音（含闪念）强制使用耳机麦克风
- **信息架构**：方案 B（两级结构）—— Search Devices 负责发现/绑定，设备控制台负责全链路管理
