这版 Demo 对齐点

  • 无 Day/Timeline 视图切换，每一天即独立时间轴。
  • `Events` 用时间流 block 展示区间事件。
  • 右上角图标切换：日历容器 / 全量 reminders 列表。
  • 日历容器内保留当日 `Events/Reminders` 切换。
  • 空白时间区点击创建 30 分钟临时“新建日程”占位。
  • 同一时间轴只保留一个临时占位；切换日期或打开其它 event 时清除。
  • 底部 `+` 为唯一新建入口，无“默认一小时”话术。
